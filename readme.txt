# YOLOv8 PCB Detection Project

Follow these steps to set up the environment and run the Jupyter Notebook.

## Steps to Run

1. Open a terminal or Anaconda prompt.

2. Navigate to the project folder:
	cd <path-to-your-project-directory>
3. Create and activate the Conda environment:
	conda env create -f environment.yml conda activate yolov8_env
4. Start Jupyter Notebook:
	jupyter notebook
5. Open the following link in your browser to access the notebook:
	http://localhost:8888/notebooks/Test%20PCB.ipynb
6. Run the notebook by executing the cells sequentially.

## Notes

- Ensure that the `Test PCB.ipynb` file is in the project folder.
- Ensure the `environment.yml` file is used to set up the required dependencies.

Enjoy!
